# Healthcare_Analysis-

## Description:
* This project analyzes healthcare data to provide insights into patient spending, departmental performance, and treatment costs. The dashboard highlights high-value patients (above 10,000 spend), average length of      stay, gender-based cost distribution, and department-wise revenue trends.
* Built using Power BI, DAX, and Data Modeling, this project demonstrates how healthcare data can be transformed into actionable business insights.
